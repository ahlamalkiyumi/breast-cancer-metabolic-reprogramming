we will filter low expression genes using CPM (Counts Per Million). CPM is used when normalizing for sequencing depth to compare relative expression levels between different samples. It normalizes each sample by its own total counts.

CPM (Counts Per Million) = (raw count / total counts in that sample) * 1,000,000

n_normal = sample["tcga.cgc_sample_sample_type"].astype(str).str.contains(
    "Solid Tissue Normal", case=False, na=False
).sum()


print(f"\nNumber of normal samples: {n_normal}")

Number of normal samples: 112
 
# Step 1: convert raw counts to CPM
lib_sizes_for_cpm = data.sum(axis=0)                     # total counts per sample
cpm = data.div(lib_sizes_for_cpm, axis=1) * 1e6           # genes x samples, now in CPM
 
# Step 2: set thresholds
min_cpm = 1                                  # CPM threshold for "expressed"
min_samples = max(10, n_normal // 2)         # same logic as before: tied to smaller group
 
# Step 3: apply filter on CPM values, but keep the ORIGINAL raw counts
# for DESeq2 -- CPM is only used here to decide which genes to keep,
# DESeq2 still needs untouched raw integer counts to run correctly.
keep = (cpm >= min_cpm).sum(axis=1) >= min_samples
data_filtered = data.loc[keep]
 
print(f"\nFiltering: keep genes with >= {min_cpm} CPM in >= {min_samples} samples")

import numpy as np
from sklearn.decomposition import PCA
import matplotlib.pyplot as plt
import seaborn as sns

log_counts = np.log2(data_filtered.T + 1)
 
pca = PCA(n_components=2)
coords = pca.fit_transform(log_counts)
 
plot_df = pd.DataFrame(coords, columns=["PC1", "PC2"], index=data_filtered.columns)
plot_df["sample_type"] = sample.loc[plot_df.index, "tcga.cgc_sample_sample_type"].values
 
plt.figure(figsize=(8, 7))
sns.scatterplot(data=plot_df, x="PC1", y="PC2", hue="sample_type", s=15, alpha=0.6)
plt.title("PCA — tumor vs. normal (QC check)")
plt.xlabel(f"PC1 ({pca.explained_variance_ratio_[0]*100:.1f}%)")
plt.ylabel(f"PC2 ({pca.explained_variance_ratio_[1]*100:.1f}%)")
plt.tight_layout()
plt.savefig("PCA_tumor_vs_normal.png", dpi=300, bbox_inches="tight")
plt.show()

The orange cluster (normal tissue) sits distinctly separate from most of the blue (tumor) cluster along PC1, that’s exactly what we want to see. It means tumor and normal samples are genuinely different at the whole-transcriptome level.
